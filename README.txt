i.   Penjelasan Singkat Algortima Decrease and Conquer

ii.  Requirement Program
        >> Pastikan Anda sudah memiliki instalasi Python 3 terbaru.
        Informasi lebih lanjut dapat dilihat di https://www.python.org/
        Setelah itu, program dapat dijalankan melalui IDLE atau
        terminal.


iii. Cara Menggunakan
     >> Menjalankan Program
        a. Menggunakan IDLE
           Klik kanan file "13519063.py" pada folder src dan pilih "Edit
           with IDLE." Lalu, tekan F5 atau pilih Menu Run -> Run Module.
        b. Menggunakan Terminal
           Lakukan navigasi ke dalam folder src, lalu masukkan perintah
           "python 13519063.py" tanpa tanda kutip.
           
     >> Memasukkan Input
        Masukkan path file input. Jika "13519063.py" dibuka dari folder src,
        maka path file adalah "../test/XX.txt" tanpa tanda petik, dengan
        XX diganti dengan angka sesuai nama file yang tersedia.
        
     >> Format File Input
        File input ditulis dengan huruf kapital dengan format seperti
        contoh berikut.
        C1.
        C2, C1.
        C3, C2, C1.
        Bahasa Inggris.
        
        (Keterangan: Mata kuliah yang ditulis pertama adalah nama matkul
                     tersebut, dan diikuti oleh semua mata kuliah lain
                     yang merupakan prerequisite-nya. Setiap mata kuliah
                     dab prerequisite-nya dipisahkan dengan ", " serta
                     diakhiri dengan tanda titik dan newline.
                     
     >> Pengaturan
        Terdapat opsi untuk membatasi jumlah mata kuliah yang diambil
        tiap semester. Untuk menggunakannya, ganti nilai variabel
        maksPerSemester pada file source code sesuai dengan jumlah
        mata kuliah maksimal per semester. Nilai nol menandakan tidak
        adanya batasan maksimal mata kuliah per semester.
